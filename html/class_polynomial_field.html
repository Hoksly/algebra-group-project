Detailed Description

template<typename T>
class PolynomialField< T >

A class representing a polynomial field.

The PolynomialField class represents a polynomial field defined over a prime modulus. It provides operations for arithmetic, exponentiation, irreducibility checking, normalization, and more.

Template Parameters
TThe type of coefficients in the polynomial.
