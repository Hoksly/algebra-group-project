template<typename T >
bool isPrimeSimpleFunction ( T  num)

Checks if a number is prime using a simple primality test.

This function checks if a number is prime by iterating from 2 to the square root of the number and checking if the number is divisible by any of the iterated values. If the number is divisible by any value in the range, it is considered non-prime.

Parameters
numThe number to be checked for primality.
Returns
True if the number is prime, otherwise false.
