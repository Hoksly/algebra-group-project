static constexpr size_t pi_size_t ( )
staticconstexpr

Returns a size_t value representing an approximation of pi.

This function returns a size_t value that approximates the mathematical constant pi. The value returned depends on the size of the size_t type. If the size of size_t is 4 bytes (32 bits), the returned value represents floor(pi/4 * 2^32). If the size of size_t is 8 bytes (64 bits), the returned value represents floor(pi/4 * 2^64).

Returns
The approximate value of pi represented as a size_t.
Exceptions
std::logic_errorIf the size of size_t is neither 4 nor 8 bytes, indicating an unsupported platform.
